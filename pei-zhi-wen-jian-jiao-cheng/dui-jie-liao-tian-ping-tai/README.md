# 📪 对接聊天平台

目前，我们的一个程序实例只能对接一个聊天平台。&#x20;

对于选择的平台 为 4 选 1，即 mirai onebot telegram discord 只能选择一个。

也就是说 `[mirai]` `[telegram]` `[onebot]` `[discord]` 只能存在一个。

## 对接 QQ

目前支持两种对接 QQ 的协议，分别为 Mirai 和 OneBot。

你可以选择你喜欢的协议进行对接。

# 📪 对接聊天平台

我们现在支持同时接入多种平台，你只需要在配置文件 `config.cfg` 加入你想要接入的平台的配置，就可以开启相关功能。

## 对接 QQ

目前支持两种对接 QQ 的协议，分别为 Mirai 和 OneBot。

你可以选择你喜欢的协议进行对接。
